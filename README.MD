### The 7 - 1 Pattern Directory Structure

```
sass/
|
|– utils/ (or abstract/)
|   |– _colors.scss             // Sass Colors
|   |– _functions.scss          // Sass Functions
|   |– _mixins.scss             // Sass Mixins
|   |– _helpers.scss            // Sass Helpers
|   |– _placeholder.scss        // Sass Helpers
|   |– _variables.scss          // Sass Variables
|
|– base/
|   |– _reset.scss              // Reset/Normalize
|   |– _typography.scss         // Typography rules
|
|– components/ (or modules/)
|   |– _buttons.scss            // Buttons
|   |– _carousel.scss           // Carousel
|   |– _inputs.scss             // Inputs
|   |– _cards.scss              // Cards
|
|– layout/
|   |– _grid.scss               // Grid system
|   |– _table.scss              // Table 
|   |– _header.scss             // Header
|   |– _footer.scss             // Footer
|   |– _sections.scss           // Sections
|   |– _sidebar.scss            // Sidebar
|   |– _forms.scss              // Forms
|
|– pages/
|   |– _home.scss               // Home specific styles
|   |– _about.scss              // About specific styles
|   |– _contact.scss            // Contact specific styles
|
|– themes/
|   |– _default.scss            // Default theme
|   |– _light.scss              // Light theme
|   |– _dark.scss               // Dark theme
|
|– vendors/
|   |– _bootstrap-grid.scss    // Bootstrap Grid
|   |– _jquery-ui.scss         // jQuery UI
|   |– _normalize.scss.scss    // jQuery UI
|
`– main.scss        

```
